# HeroGym

HeroGym is a Java-based gym management application.

## Features

* Member Management
* Coach Management
* Course/Class Management
* Database Integration (using MS Access and UCanAccess)

## Requirements

* Java Runtime Environment (JRE) / Java Development Kit (JDK)
* UCanAccess dependencies (included in the `lib` folder)

## Setup and Installation

1. Clone or download the repository.
2. Open the project in your preferred Java IDE (such as Apache NetBeans, which is supported out of the box via the `nbproject` directory).
3. Ensure the required UCanAccess JAR files are properly added to the project's library path.
4. Build and run the project.

## Development

This project uses Ant for its build system (`build.xml`).
